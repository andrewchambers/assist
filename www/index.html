assist

AI-powered command line assistant for Unix systems

Automate tasks, fix bugs, and get things done with natural language

Install Quick Start Manual Source Code Discord 
$ # What is assist?
$ assist "What are you and what can you do?"

I'm assist, an AI-powered command line tool that helps you accomplish tasks on Unix systems.

Here's what I can do:

  • System administration - Find processes, manage services, debug network issues
  • File management - Organize files, batch rename, find duplicates, clean up disk space
  • Fix bugs - Diagnose compilation errors and apply fixes automatically
  • Refactor code - Modernize, optimize, or restructure your codebase
  • Automate tasks - Create scripts, set up cron jobs, build workflows
  • Data processing - Parse logs, extract information, convert between formats

I maintain focus on relevant files and execute commands in a controlled environment
until your task is complete.

$ # See it in action
        
$ # Installation
$ git clone https://github.com/andrewchambers/assist.git
$ cd assist
$ make
$ sudo make install PREFIX=/usr/local
Installing assist to /usr/local/bin/
Installing man pages to /usr/local/share/man/
        
$ # Quick Start - Choose your model provider

$ # Option 1: OpenRouter (access to many models)
$ export OPENROUTER_API_KEY="your-key-here"
$ assist "Fix the memory leak in util.c"

$ # Option 2: OpenAI
$ export OPENAI_API_KEY="your-key-here"
$ assist --model o3-pro "Refactor this function to be more efficient"

$ # Option 3: Local models with Ollama
$ # First configure ~/.config/assist/models.json (see docs)
$ assist --model qwen3:32b "Explain this codebase structure"
        
$ # Real-world examples

$ # Fix compilation errors
$ assist --focus '*.c' "Fix all compilation errors in this C project"

$ # Add a new feature
$ assist --focus 'src/*.js' "Add dark mode support to the UI"

$ # Refactor code
$ assist "Convert all callbacks to async/await in the API module"

$ # Debug issues
$ assist "Find which process is listening on port 8080"

$ # Generate documentation
$ assist "Document all public functions in the math module"
